Proyek ini merupakan tugas kelompok berupa perancangan dan implementasi
database untuk Sistem Informasi Manajemen Nikah (SIMKAH) menggunakan MySQL.

## Deskripsi Proyek
SIMKAH dirancang untuk mengelola data administrasi pernikahan secara terstruktur,
meliputi data calon pengantin, jadwal pernikahan, petugas, dan pencatatan peristiwa
nikah.

## Peran Saya dalam Proyek
- Berkontribusi dalam perancangan struktur tabel dan relasi database
- Melakukan pengolahan dan input data ke dalam MySQL
- Membantu pengujian query serta validasi konsistensi data

## Alur Pengolahan Data
Data awal disiapkan dan dibersihkan menggunakan Microsoft Excel,
kemudian diimpor ke dalam database MySQL untuk proses perancangan
dan pengelolaan data.

## Teknologi yang Digunakan
- MySQL
- Microsoft Excel

## Catatan
Proyek ini merupakan hasil kerja kelompok sebagai bagian dari tugas perkuliahan.
